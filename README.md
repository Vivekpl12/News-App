# Flutter News App

A Flutter news application demonstrating REST API integration, JSON parsing, clean architecture, and async UI handling.

## Features

- Fetch top headlines from NewsAPI
- REST API integration using http package
- JSON parsing with model classes
- Repository architecture
- Loading, success, and error states
- Retry failed requests
- Pull-to-refresh
- Responsive card UI

## Architecture

lib/
- models → data models
- services → API layer
- repository → data abstraction
- screens → UI screens
- widgets → reusable components
